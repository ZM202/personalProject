Let's get started on learning Javascript.
You might be asking what is Javascript?

  • JavaScript is the Programming Language for the Web. You can change how the page behaves, adding allows users to interact with web pages.
  • There are almost no limits to the things you can do with JavaScript on a web page.
  • Here are a few things you can do with javascript. via hackreactor....

    • Show or hide more information with the click of a button
    • Change the color of a button when the mouse hovers over it
    • Slide through a carousel of images on the homepage
    • Displaying a timer or count-down on a website
