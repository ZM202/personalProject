What is Variables? In order for the system to do its job, the data needs to be store somewhere. That is when the variables came in. The data can be store in variables.
First, you need to declare a variable. You can use var, let, or const. Most developers will use either let or const to declare a variable. const is the same as let variable except, it cannot be reassigned.

Let's take a look at some examples to how to declare a variables.
let >> variable keyword
grocery>> variable name
avocado>> variable value
let grocery = avocado;

What is Data Types and how do we incorporate into coding? Data Types are important since it allow programmers to store different types of data. You can have string, boolean,numbers, undefined, null, object, and array data types. Let’s tackle on how to write each data types using a variable to store a data.....

The string data type can use either single or double quotes. 
let grocery = “avocado”;
let grocery = 'avocado';

The boolean data type can have two values of either true or false.
let fruit = orange;
let vegetable = iceCream;
fruit = true;
vegetable = false;

The numbers data type can be written as with or without decimals.
let x = 5;
let y = 5.00;

The undefined data type is when a variable has been declared, but has not been assigned a value.
let x;
let y = "fruits";
console.log(x);
                
The output will be undefined since there is no value assigned to x.
                
console.log(y);
                
The output will be fruits since there is a value assigned to y as fruits.

The null is when value does not exist. In Javascript, null is an object.
let x = null;
console.log(x);
The output will be null since it is assigned as null.

Arraymark> used for storing multiple values in single variable using square brackets and separated by using commas.
let fruits = ['grapes', 'oranges', 'apples'];

Objects allows to store collections of data by using curly brackets. An object contains properties which are defined as key value pairs.
let fruits = {
    name = "apples";
    cost = 1;
};

In this example, an object represents fruits. name and cost represent keys. The apples and 1 represent values.
